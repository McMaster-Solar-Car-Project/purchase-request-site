# Product Overview

Purchase Request Site is a FastAPI web application designed to streamline purchase request and expense report management for the McMaster Solar Car Project. The primary goal is to help team members quickly get reimbursed for project-related expenses.

## Key Features
- Purchase request submission and tracking
- Expense report management
- Google Sheets integration for data storage
- Email notifications for request processing
- User authentication and session management
- File upload and processing capabilities
- Rate limiting for security

## Target Users
- McMaster Solar Car Project team members
- Operations and Finance teams (administrators)

## Business Context
This is an internal tool for a university solar car team, focused on simplifying financial processes and reducing administrative overhead for expense reimbursements.
